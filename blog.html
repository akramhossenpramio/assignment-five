Question-1: What are the different ways to select an element in the DOM?

Use getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), or querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets/sets HTML content, innerText gets/sets only visible text, and textContent gets/sets all text including hidden content.

Question-3: What is event delegation in the DOM?

It is a technique where a parent element handles events for its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

An event on a child element propagates to its parent and ancestors. It can be stopped using event.stopPropagation().

Question-5: How do you create, add, and remove elements using JavaScript?

Use document.createElement() to create, appendChild() to add, and remove() or removeChild() to delete elements.
